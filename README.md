# SeniorAlarm

## 📄 Introduction
정해진 시간에 무언가를 해야하는 독거노인을 위한 알람 어플

## 🔑 Features
- 정해진 시간에 음성으로 입력해놓은 메세지가 알람으로 울림
- 일정 시간 후 음성으로 수행 여부 확인 및 응답 녹음
- 사용자가 응답하여 중지하기 전까지 일정 간격으로 반복 확인

## 📱 Test Environments
```
Android 8.0
```

## 💻 Framework and Dependency
```
Google Cloud Platform Text-To-Speech
```
